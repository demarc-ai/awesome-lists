- create-next-app, create-vite, create-expo, create-t3-app
- degit, turbo, nx, changesets

Recommended:
- create-next-app — Official Next.js starter.  
  npm: next, starter: https://nextjs.org/docs/app/api-reference/create-next-app
- create-vite — Minimal scaffolding for modern web apps.  
  npm: create-vite
- create-expo — Expo React Native starter.  
  npm: create-expo-app
- create-t3-app — Full-stack TS (Next.js + Prisma + tRPC + Tailwind).  
  npm: create-t3-app
- degit — Straight copy git repo boilerplates (no history).  
  npm: degit
- turbo / nx — Monorepo toolchains; choose Turbo for speed, Nx for generators.  
  npm: turbo, nx
- changesets — Versioning and changelogs for monorepos.  
  npm: @changesets/cli

### 🏗️ Build Tools
Bundlers, compilers, and dev servers

From awesome-npm-complete (merged):
- vite, webpack, rollup, esbuild, swc, parcel, rspack

Recommendations:
- Vite — Best default DX for modern apps.  
- esbuild — Fastest builds; pair with tsx/tsup.  
- SWC — Rust TS/JS compiler; used by Next.js.  
- Rspack — Rust-based webpack alternative focused on speed.  
- Rollup — Libraries-focused bundling (tree-shaking).  
- Webpack — Enterprise flexibility, legacy compat.
